# Deep Learning & Neural Networks — Practice Workbook

Interactive study workbook built from the **SST "Deep Learning I" (Neural Networks & CV)** course handouts —
18 sessions across 8 weeks. Each session has **short revision notes**, an **interactive self-grading quiz**
(MCQ with explanations), and **descriptive questions with model answers**. Math is rendered with KaTeX. No coding questions.

Content: 151 note sections · 216 quiz questions · 144 descriptive Q&A — all in `src/content.json`.

## Run

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build -> dist/
npm run preview  # serve the build
```

Quiz answers and revealed solutions persist in `localStorage` per session.
